# TF - DW - Aula 03

Trabalho final da aula 03 de Desenvolvimento Web utilizando Node.js.

## Como executar

No terminal:

node tf03.js

## O que o código demonstra

Uso de objetos, arrays dentro de objetos, loops (for...in e for...of), comparação de tipos de referência, cópia por referência, cópia com spread operator e conversão para JSON.
